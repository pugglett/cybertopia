How to Create a Website.

How to use various features in HTML

Creating a basic website is simple and fun! Once you learn the basics you're on your way to creating masterpieces. Here are some key things needed to create a successful basic webpage.

  • First thing is first when creating a website: save your document as index.html in a fresh new folder.

    This allows your text editor to know that you are now creating a webpage, makes it easier to write your code, as well as see it in an internet browser.

  • Once your page is saved the first thing you want to type in the editor is < !DOCTYPE html>.

    This allows the text editor to understand that now you are creating an HTML website. From this point forward you can now add HTML elements.

  • All page elements are placed between < html> ... < /html>.

    This is known as the root element. All other elements are known as children to this element.

  • Next you'll need your < head> ... < /head> element.

    This only appears once in an HTML document and provides the general information about the document.

  • Once you have your head element you will enter the < title> ... < /title> element.

    This is a child of the head element and it defines the title of the document. This is shown on the browser's title bar.

  • Now you're ready for the < body> ... < /body> element.

    There can only be one < body> element in a document. The body represents the content of an HTML document.

  • To add topics to your website you will use the heading element < h1> ... < /h1> - < h6> ... < /h6> .

    There are six different levels, < h1> is the most important and < h6> is the least. You're only allowed one < h1> tag a page.

  • When you are ready to enter your text for the website the paragraph element < p> ... < /p> is how you separate your text.

    Text can range from one sentence to full paragraphs.

  • If you want to add lists to your website you can do so in two different ways.

    In order to get a bulleted list (unordered) you want to use the < ul>...< /ul> tag. If you would like your list to be numbered (ordered) the < ol>...< /ol> tag will be used. To represent the items on a list < li>... < /li> tag is used.

These are just the basic elements of an HTML document. There are many more elements that tie into creating the websites that you use on a day to day basis. As you become more advanced in your HTML knowledge you will begin to learn how to use CSS and relate it into the styling of your website.
